# Magic Road 🎮

An **interactive road-drawing game** built with **Unity (C#)**, where players dynamically create paths to influence gameplay in real time.

---

## Features

- ✏️ Draw roads freely during gameplay to dynamically control the car’s path  
- 🎮 Real-time interaction — your input directly affects movement and helps navigate obstacles  
- 🚗 Collect coins to unlock a variety of cars

---

## Tech Stack

- **Engine:** Unity  
- **Language:** C#  

---

## How to Run

1. Clone the repository:

        git clone https://github.com/shjy2008/DrawRoadGame.git

2. Open the project in **Unity Hub**

3. Click **Play** in the Unity Editor
